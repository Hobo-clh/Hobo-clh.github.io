单链表实现栈

思路

使用链表模拟栈
思路：
入栈：当要插入数据时，直接将其放在头节点之后
newNode.next = head.next; head.next = newNode;
出栈：删除返回头节点的下一个节点，
StackNode data = head.next;
head.next = head.next.next;
return data.value;
遍历：list(),从头节点开始遍历
